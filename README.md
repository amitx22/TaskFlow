# 🚀 TaskFlow  

## 📌 Overview  
TaskFlow AI Pro is a modern, responsive, and feature-rich task management web application built using **HTML, CSS, and JavaScript**. It helps users efficiently organize daily tasks with features like task creation, deadline tracking, priority management, search, filtering, progress analytics, and local storage support.

Designed with a sleek **glassmorphism UI**, this project combines productivity with an attractive user experience.

---

# ✨ Features  

## 📝 Task Management  
- Add new tasks instantly  
- Set due dates for deadlines  
- Assign priority levels:
  - 🔴 High
  - 🟠 Medium
  - 🟢 Low  
- Edit tasks anytime  
- Delete tasks  
- Mark tasks as completed  

## 🔍 Search & Filter  
- Real-time search functionality  
- Filter tasks by:
  - All Tasks  
  - Active Tasks  
  - Completed Tasks  

## 📊 Productivity Dashboard  
- Total Tasks Counter  
- Pending Tasks Counter  
- Completed Tasks Counter  
- Completion Percentage Tracker  

## 💾 Local Storage Support  
- Saves tasks in browser storage  
- Data remains after refresh or reopening browser  

## 📱 Responsive Design  
- Mobile-friendly  
- Desktop optimized  
- Adaptive layout
